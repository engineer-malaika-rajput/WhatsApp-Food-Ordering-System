# WhatsApp Food Ordering System

An automated food ordering system built with n8n that allows customers to place orders through WhatsApp while automating order processing, notifications, and workflow management.

## Features

* WhatsApp-based food ordering
* Automated order processing
* Menu management
* API & Webhook integration
* Customer notifications
* Workflow automation using n8n

## Tech Stack

* n8n
* WhatsApp Business API
* Webhooks
* REST APIs
* Database Integration

## Learning Outcomes

This project provided practical experience in workflow automation, API integrations, WhatsApp business solutions, and real-world deployment challenges.

## Future Improvements

* AI-powered order recommendations

* Admin dashboard
* Multi-restaurant support
